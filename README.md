# Dashboard UESC - Custo de Oportunidade Agropecuário

Arquivos para publicar no GitHub/Streamlit Community Cloud.

## Arquivos na pasta

- `app.py`
- `requirements.txt`
- `BASEFINAL_mun_sem_menores_5ha_sem_outliers_acima_15000.csv`
- `Brasão_da_UESC.png`

## Rodar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Publicar

1. Crie um repositório no GitHub.
2. Envie os quatro arquivos acima.
3. Acesse https://share.streamlit.io.
4. Escolha o repositório e selecione `app.py`.
5. Clique em Deploy.

## Observação

O mapa municipal pode demorar. Use filtros por UF/microrregião ou marque `Tentar carregar mapa municipal completo`.
